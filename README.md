# notes

A command line tool for managing notes, their respective categories, and templates (basic notes, memos, etc).

**/!\ This program is just at its beginning stage, it's not usable!**

## Presentation

`notes` is a program written in C++ and Python, it offers a handy command line tool for managing your notes.

## Basic Usage

**New Notes**

To create a new note:

```bash

notes note create "Title of my note" -c "Personal"
```

**List Notes**

To list all notes in the "Personal" category:

```bash
notes note list -c "Personal"
```
